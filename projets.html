Mes Projets

Site Responsive

Site web moderne conçu pour offrir une expérience utilisateur optimale sur tous les types d’écrans (ordinateurs, tablettes et smartphones).

Le design s’adapte automatiquement grâce aux techniques de responsive design (flexbox, media queries), garantissant une navigation fluide et agréable.

Technologies : HTML5, CSS3 (Flexbox, Media Queries)

App JavaScript

Application web interactive développée en JavaScript, mettant en œuvre des animations dynamiques et une interface utilisateur fluide.

Elle permet aux utilisateurs d’interagir en temps réel avec les éléments de la page (événements, manipulation du DOM, effets visuels).

Technologies : JavaScript (ES6+), HTML5, CSS3

Gestion PHP

Application de gestion développée en PHP avec une base de données MySQL, permettant de stocker, organiser et manipuler efficacement les données.

Le système inclut des fonctionnalités comme l’ajout, la modification, la suppression (CRUD), ainsi qu’une interface sécurisée pour les utilisateurs.

Technologies : PHP, MySQL, HTML5, CSS3
